# Multiplication

In JavaScript, we can perform the multiplication of two numbers by using the asterisk `(*)` arithmetic operators.&#x20;

### Example:

```javascript
let resultingValue = 3 * 2;
```

Here, we stored the product of `3 * 2` into a `resultingValue` variable.
