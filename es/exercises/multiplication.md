# Multiplicación

En JavaScript, podemos realizar la multiplicación de dos números usando el operador aritmético asterisco `(*)`.

## Ejemplo

```javascript
let valorResultante = 3 * 2;
```

Aquí, hemos almacenado el producto de `3 * 2` en la variable `valorResultante`.
