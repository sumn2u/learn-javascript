# Perkalian

Dalam JavaScript, kita dapat melakukan perkalian dua angka dengan menggunakan operator aritmatika asterisk `(*)`.&#x20;

### Contoh:

```javascript
let nilaiHasil = 3 * 2;
```

Di sini, kita menyimpan hasil dari `3 * 2` ke dalam variabel `nilaiHasil`.
